# 对生成图片的具体要求（必读）：
1. 必须是单张、高清晰度、无水印的设计稿图片。
2. 视觉风格要符合企业级中后台标准（例如： Element Plus 或 Ant Design 风格）。
3. 布局要合理：例如，左侧是可折叠的筛选栏，右侧是数据看板和表格。
4. **核心要求（用于后续自动化）**：图片必须包含清晰的文本信息（例如：按钮文字“新增跟进”、卡片标题“本月跟进数”、表格列名“跟进时间”、“操作人”）。这将用于后续让 Trae 进行 OCR（光学字符识别）还原。
5. 不要生成多张小图合并或多角度展示，只需一张清晰的平面 Mockup 即可。
